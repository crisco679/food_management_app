#Food Management App

Keep track of your monthly food budget!

## Requirements
* Node.js (v6.2.1)

## Instructions
1. `git clone https://github.com/crisco679/food_management_app.git`
2. `cd food_management_app`
3. `npm install`
4. `npm start`
